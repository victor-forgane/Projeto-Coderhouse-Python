# **Projeto: Extração e Validação de Dados da API Pokémon**

Este projeto foi desenvolvido como parte de uma entrega parcial para o curso de Python. Ele realiza a extração de dados de uma API pública ([PokéAPI](https://pokeapi.co/)), trata as informações, armazena em um banco de dados SQLite e as valida para consulta posterior.

---

## **Objetivo do Projeto**
Criar um pipeline que:
- Extrai dados de múltiplas tabelas da [PokéAPI](https://pokeapi.co/).
- Realiza o tratamento dos dados para estruturar em tabelas.
- Adiciona os dados tratados a um banco de dados SQLite.
- Valida as tabelas disponibilizadas no banco de dados.
- Implementa um sistema de alerta para identificar inconsistências.

---

## **Requisitos do Projeto**

1. Escolher uma API da lista disponível.
2. Fazer a extração de pelo menos **3 tabelas**.
3. Realizar os tratamentos necessários para organizar os dados.
4. Adicionar os dados tratados em um banco de dados para consulta.
5. Validar as tabelas disponibilizadas.
6. Criar um ambiente virtual para o projeto.
7. Disponibilizar o projeto no GitHub.
8. Criar um sistema de alerta.
